------------------------------- What is Figma ------------------------------------
- Figma is a cloud-based UI/UX design and prototyping tool used to create website designs, mobile app layouts, wireframes, and interactive prototypes.

- It works directly in the web browser, so designers can design, share, and collaborate in real time without installing heavy software. 


----------------------------------- importance of Figma -------------------------------
1. Real-time collaboration – Multiple designers and developers can work on the same design at the same time.
2. Cloud storage – Designs are saved online, so no risk of losing files.
3. Easy sharing – You can share a simple link for feedback or developer handoff.
4. Prototyping feature – Create clickable and interactive UI without coding.
5. Component & reusable design system – Helps maintain consistency in large projects.
6. Cross-platform – Works on Windows, Mac, and even in the browser.
7. Free plan available – Beginners can start learning without paying.

Figma makes designing, collaboration, and sharing faster and easier, which is why most modern UI/UX designers prefer it.

**** In web design, vector and raster images are used for different purposes. Here’s a clear, easy-to-understand comparison:

1. Vector Images : Images made using mathematical formulas (lines, curves, shapes).

Examples: SVG, AI, EPS

Features:
1. Scalable (zoom in/out without losing quality)
2. Small file size (usually lighter)
3. Best for logos, icons, illustrations

Use in Web Design:
1. Logos
2. Icons
3. UI elements (buttons, shapes)

2. Raster Images : Images made of pixels (tiny dots).
Examples: JPEG, PNG, GIF, WebP

Features:
1. Best for photographs and detailed images
Note: Not scalable (loses quality when zoomed), Larger file size (depends on resolution)

Use in Web Design:
1. Photos
2. Background images
3. Product images

----------------------- Alignment in Figma ----------------
- Alignment means arranging elements in a straight line or proper position so your design looks clean and organized.

Types of Alignment
1. Horizontal Alignment : Align Left, Align Center, Align Right
2. Vertical Alignment : Align Top, Align Middle, Align Bottom

Why Alignment is Important
- Makes UI look clean & professional
- Improves readability
- Keeps design consistent

Note: Alignment Layer to parent
- align layers in relation to each other 
- align left, right, horizon, center, top , bottom and vertical 
- Hold Shift to select multiple layers


Windows Short - key for Alignment 
1. Align Top : Alt + W 
2. Align Left : Alt + A
3. Align bottom : Alt + S
4. Align Right: Alt + D
5. Align Vertical centers: Alt + V
6. Align Horizontal Centers: Alt + H

----------------------------- plugins ---------------------
List of Plugins we used in Figma 

// Image & Content Plugins
1. Unsplash : Unsplash plugin for Figma is a Figma plugin that allows designers to insert high-quality, free stock images directly into their designs. It saves time by providing instant access to a large library of professional photos without leaving Figma.

2. Content Reel : Adds dummy text, names, emails, and images for UI design.

// Icons & UI Elements
1. Blush: Custom illustrations for websites and apps.

2. iconify : Iconify plugin for Figma is a plugin that gives access to thousands of icons from different icon libraries. Designers can easily search, customize, and add icons to their UI designs in just a few clicks.

//Layout & Alignment Position
1. Sorter : Sorter plugin for Figma is a plugin used to automatically arrange and organize layers or objects in a structured order (like grid or alignment). It helps maintain consistency and improves layout design efficiency.

2. Auto Layout : Helps create responsive designs automatically.

//Color & Design System
1. Coolors : Generates beautiful color palettes instantly.

2. Design Lint : Finds and fixes inconsistent styles in your design.

// Testing & Developer Handoff
1. Stark : Checks accessibility (color contrast, vision issues).
2. Zeplin : Helps developers understand design specs easily.

// Productivity Boosters
1. Remove BG : Removes image background automatically.
2. Figmotion : Create animations inside Figma.
