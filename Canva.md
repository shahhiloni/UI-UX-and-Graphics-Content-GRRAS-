1. Introduction to Canva
- Canva is a graphic design tool used to create:
1. Social media posts
2. Presentations
3. Posters & flyers
4. Resumes
5. Videos

- It works online (browser) + mobile app
- No design experience required

2. Canva Interface 

- Home Dashboard → Templates + recent designs
- Editor Page → Where you design

Left Panel:
- Templates
- Elements
- Text
- Uploads

Top Bar:
- Resize
- Share
- Download

3. Basic Tools (Must Know)
- Text
- Add headings, subheadings, body text

Customize:
Font, Size, Color, Alignment

4. Elements
- Shapes, icons, stickers, illustrations

Search anything like:
“business icons”
“social media”

5. Colors
Use:
1. Brand colors
2. Gradient backgrounds
3. Copy color using eyedropper tool

*** Uploads images, vidoes, logos

6. Layout & Design Principles
- Alignment → Keep elements straight
- Spacing → Avoid clutter
- Contrast → Make text readable
- Hierarchy → Important things bigger

7. Working with Templates
- Canva provides 1000+ ready templates

Steps:
1. Search template (e.g., Instagram post)
2. Replace text & images
3. Customize colors
