# CorelDRAW Notes (Beginner Friendly)
## 1. What is CorelDRAW?
- CorelDRAW is a vector graphics design software used to create logos, banners, posters, brochures, visiting cards, and illustrations.

## 2. Features of CorelDRAW

* Vector-based software (no quality loss on zoom)
* Easy to use interface
* Used for printing and designing work
* Supports multiple file formats

## 3. Interface (Basic Layout)

- Menu Bar** → Top (File, Edit, View, Layout, etc.)
- Toolbox** → Left side (Pick Tool, Shape Tool, etc.)
- Property Bar** → Top (changes based on selected tool)
- Drawing Area (Page)** → Where you design
- Color Palette** → Right side for colors

## 4. Important Tools (Must Know)
1. Pick Tool
- Used to select and move objects

2. Shape Tool
- Used to edit shapes and nodes

3. Bezier Tool / Pen Tool
- Used to draw custom shapes and curves

4. Rectangle Tool
- Used to draw rectangles and squares

5. Ellipse Tool
- Used to draw circles and ovals

6. Text Tool
- Used to add text

7. Fill Tool
- Used to fill colors in objects

8. Interactive Fill Tool
- Used to apply gradient colors

5. What are Vectors?

- Graphics made using lines and curves
- Do not lose quality when resized
- Used in logo and printing designs

6. Object Properties

- Fill → Inside color
- Outline → Border of object
- Size & Position → Control object dimensions

7. Layers (Objects Management)

- Used to organize objects
- Helps in easy editing
- Objects can be arranged forward/backward

8. File Formats

- .CDR → CorelDRAW file
- .PNG → Transparent image
- .JPG → Normal image
- .PDF → Print format

9. Important Tips

- Use vector shapes for better quality
- Organize objects properly
- Use alignment tools
- Practice regularly

10. Conclusion
- CorelDRAW is widely used for graphic designing and printing work.
With practice, you can create professional designs easily.

Practice daily to improve your design skills
