---------------------- Adobe Illustrator (Vector Design)
1. What is Illustrator?
- Adobe Illustrator is a vector graphics software used to create logos, icons, illustrations, posters, and designs that can be resized without losing quality.

Used for: Logos, Icons, Illustrations
- Works on vectors (no quality loss)


2. Basic Tools
- Selection Tool
- Pen Tool (MOST IMPORTANT)
- Shape Tool
- Text Tool
- Pen Tool (Key Skill)

3. Create custom shapes
- Draw curves and lines
- Colors & Stroke
- Fill → Inside color
- Stroke → Border
- Pathfinder (Shape Editing)
- Combine shapes
- Cut shapes

4. Layers & Artboards
- Multiple designs in one file
- Organize elements
- Advanced Illustrator
- Gradient Mesh
- 3D Effects
- Typography design
- Logo creation

5. Use Cases
- Logo design
- Branding
- Icon sets
- Illustrations

6. Difference Between Photoshop & Illustrator
- Photoshop → Works on pixels (raster images)
- Illustrator → Works on vectors (scalable graphics)

Example: Logo design is best in Illustrator

7. Interface 
- Menu Bar → Top (File, Edit, Object, etc.)
- Toolbar → Left side (Selection, Pen, Shape tools)
- Properties Panel → Right side settings
- Artboard → Workspace where you design

8. Important Tools (Must Know)
- Selection Tool (V)
Select and move objects

- Direct Selection Tool (A)
Edit anchor points of shapes

- Pen Tool (P)
Create custom shapes and curves

- Shape Tools
Create basic shapes (Rectangle, Circle, Polygon)

- Brush Tool (B)
Draw freehand lines

- Type Tool (T)
Add text

- Gradient Tool (G)
Apply gradient colors

9. What are Vectors?
- Vectors are made using mathematical paths
- They do not lose quality when zoomed
- Used in logos and illustrations

10. Fill & Stroke
- Fill → Inside color of shape
- Stroke → Border/outline of shape

11. Artboard
- Artboard is like a page/canvas
- You can create multiple artboards in one file

12. File Formats
- .AI → Illustrator file
- .SVG → Scalable vector format
- .PNG → Transparent image
- .JPG → Normal image
