. Adobe Photoshop (Raster Editing)
1.  What is Photoshop?
- Adobe Photoshop ek image editing software hai jiska use photo edit, graphic design, banner, poster, social media post banane ke liye hota hai.

Used for:
- Photo editing
- Image manipulation
- Social media graphics
- Works on pixels (raster images)

- Basic Tools
1. Move Tool → Move objects
2. Crop Tool → Resize image
3. Brush Tool → Drawing
4. Eraser → Remove parts
5. Text Tool → Add text

- Layers (MOST IMPORTANT)
Each element = separate layer

Types:
1. Image layer
2. Text layer
3. Adjustment layer

Use:

1. Lock / Hide / Duplicate
2. Group layers
3. Selection Tools
4. Marquee Tool
5. Lasso Tool
6. Magic Wand

Used to:

- Cut objects
- Remove background
- Masks (Intermediate)
- Hide part of image without deleting

Very useful for:
- Background removal
- Blending images

Adjustment Tools
- Brightness / Contrast
- Hue / Saturation
- Color Balance

Advanced Photoshop
- Smart Objects
- Filters (Blur, Sharpen)
- Layer Styles (Shadow, Glow)
- Photo Retouching (skin smoothing)

Compositing (combine images)
** Use Cases
- Photo editing
- Social media design
- Banner/poster creation

2. Photoshop Interface
- Menu Bar → Top me hota hai (File, Edit, Image, etc.)
- Toolbar → Left side me tools (Move, Brush, Eraser)
- Layers Panel → Right side me layers manage karte hai
- Workspace → Beech me jaha image edit hoti hai

3. What are Layers?
Layers are like transparent sheets
Each element is placed on a separate layer
Makes editing easy and non-destructive

4. Basic Shortcuts (Important)
Ctrl + N → New File
Ctrl + O → Open File
Ctrl + S → Save
Ctrl + Z → Undo
Ctrl + T → Transform (resize/rotate)
Ctrl + J → Duplicate Layer

5. Image Editing Basics
- Brightness/Contrast : Used to make the image lighter or darker
- Hue/Saturation : Used to change colors
- Crop Tool : Used to remove unwanted parts of an image

6. Adding Text
- Select Type Tool (T)
- Click on the canvas and type text
- You can change font, size, and color

7. What are Filters?
- Filters are used to add effects to images
Examples:

- Blur
- Sharpen
- Artistic Effects

8. File Formats
- .PSD → Editable Photoshop file
- .JPG → Standard image format
- .PNG → Supports transparent background
