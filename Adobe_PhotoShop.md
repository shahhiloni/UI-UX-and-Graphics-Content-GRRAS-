. Adobe Photoshop (Raster Editing)
1.  What is Photoshop?

Used for:
- Photo editing
- Image manipulation
- Social media graphics
- Works on pixels (raster images)

- Basic Tools
1. Move Tool → Move objects
2. Crop Tool → Resize image
3. Brush Tool → Drawing
4. Eraser → Remove parts
5. Text Tool → Add text

- Layers (MOST IMPORTANT)
Each element = separate layer

Types:
1. Image layer
2. Text layer
3. Adjustment layer

Use:

1. Lock / Hide / Duplicate
2. Group layers
3. Selection Tools
4. Marquee Tool
5. Lasso Tool
6. Magic Wand

Used to:

- Cut objects
- Remove background
- Masks (Intermediate)
- Hide part of image without deleting

Very useful for:
- Background removal
- Blending images

Adjustment Tools
- Brightness / Contrast
- Hue / Saturation
- Color Balance

Advanced Photoshop
- Smart Objects
- Filters (Blur, Sharpen)
- Layer Styles (Shadow, Glow)
- Photo Retouching (skin smoothing)

Compositing (combine images)
** Use Cases
- Photo editing
- Social media design
- Banner/poster creation
